# FreelancerHub 🚀

A SaaS-based freelancer marketplace built using React, Firebase, and Tailwind CSS.

## 🌟 Features
- Role-based dashboards (Client / Freelancer)
- Real-time bidding system
- Project posting and management
- Username-based authentication (Email + Username login)
- Assigned project tracking
- Modern SaaS UI (Glassmorphism + Tailwind)

## 🛠 Tech Stack
- React.js
- Firebase (Auth + Firestore)
- Tailwind CSS
- Vite

## 📂 Modules
- Authentication System
- Client Dashboard
- Freelancer Dashboard
- Bidding System
- Project Management

## 🚀 Future Improvements
- User profile page
- Chat system between client & freelancer
- Notifications
- Payment integration
